# Student Grade Calculator (Java)

A simple Java console-based application that calculates total marks, average percentage, and final grades for students based on multiple subjects.

## Overview
This project was built to practice Java programming fundamentals, modular programming, user input handling, and logical problem solving.

The application allows users to enter marks for multiple subjects, validates the input to ensure marks remain within the valid range (0–100), calculates the total and average marks, and assigns a final grade based on the calculated percentage.

The project also demonstrates clean code structure by separating logic into reusable methods.

## Features
• Input validation for marks (0–100)  
• Multiple subjects support  
• Automatic total marks calculation  
• Average percentage calculation  
• Grade assignment system (A, B, C, D, F)  
• Modular and beginner-friendly code structure  
• Console-based interactive program

## Grade Criteria
• A = 90 and above  
• B = 80–89  
• C = 70–79  
• D = 60–69  
• F = Below 60  

## How to Run the Project
1. Download or clone the repository  
2. Open the project in IntelliJ IDEA or any Java IDE  
3. Compile the Java file  
4. Run `StudentGradeCalculator.java`
5. Enter the number of subjects  
6. Enter marks for each subject  
7. View the final result report
